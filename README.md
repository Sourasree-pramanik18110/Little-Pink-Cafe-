# ✿ Little Pink Café ✿

An interactive aesthetic café website made using, Django, HTML, TailwindCSS, and JavaScript.

Live Website:  
https://sourasree-pramanik18110.github.io/Little-Pink-Cafe-/

---

## ❀ Features

✿ Beautiful pink café themed UI  
✿ Interactive table selection  
✿ Solo and friends seating options  
✿ Animated page slide transitions  
✿ Multiple food ordering system  
✿ Dynamic order cart  
✿ Return to table functionality  
✿ Payment page with receipt  
✿ Interactive customer review section  
✿ Review animations and thank you page  
✿ Fully responsive design  
✿ Hosted using GitHub Pages

---

## ❀ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
- GitHub Pages

---

## ❀ Café Sections

### ✿ Welcome Page
Customers enter the café experience.

### ✿ Table Selection
Choose:
- Sit Solo
- Sit With Friends

### ✿ Café Menu
Includes:
- Coffees
- Teas
- Pasta
- Desserts

Customers can:
- Add multiple items
- Order more food
- Return to table

### ✿ Payment Counter
Animated payment experience with receipt.

### ✿ Review Section
Customers can:
- Rate the café
- Write custom reviews
- Receive personalized responses

### ✿ Thank You Screen
Displays:

"✿ Thank you for reviewing Little Pink Café ❀"

with creator credit.
