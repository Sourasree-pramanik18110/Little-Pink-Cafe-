# ✿ Little Pink Café ✿

An interactive aesthetic café website made using, Django, HTML, TailwindCSS, and JavaScript.

Live Website:  
https://sourasree-pramanik18110.github.io/Little-Pink-Cafe-/
---
## ❀ Features

✿ Beautiful pink café themed UI  
✿ Interactive table selection  
✿ Solo and friends seating options  
✿ Animated page slide transitions  
✿ Multiple food ordering system  
✿ Dynamic order cart  
✿ Return to table functionality  
✿ Payment page with receipt  
✿ Interactive customer review section  
✿ Review animations and thank you page  
✿ Fully responsive design  
✿ Hosted using GitHub Pages

## ❀ Technologies Used
- HTML5
- Tailwind CSS
- JavaScript
- Django
